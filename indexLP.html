Linguagem de Programação

Conceitos

O que é?

Podemos considerar a linguagem de programação como uma grande rede de lógica, uma vez que ela permite expressar instruções e algoritmos para manipular dados e realizar diferentes tipos de funções. Existem diversas linguagens de programação disponíveis, cada uma com suas próprias características e propósitos. Alguns exemplos de linguagens populares incluem Python, Java, C++, JavaScript, Ruby e muitas outras. Cada linguagem de programação é projetada para atender a diferentes necessidades e áreas de aplicação. Por exemplo, algumas linguagens são mais adequadas para desenvolvimento de jogos, enquanto outras são voltadas para o armazenamento e manipulação de dados. Existem também linguagens específicas para o desenvolvimento de programas e aplicativos em geral.

Características

As características mais presentes em qualquer linguagem de programação incluem sua sintaxe, o sistema de tipagem utilizado, o paradigma de programação suportado, a disponibilidade de bibliotecas que podem existir, portabilidade entre plataformas, a eficiência e desempenho, e a existência de uma comunidade e suporte. Cada linguagem tem suas características únicas, atendendo a diferentes necessidades e preferências dos desenvolvedores.

Ambientes programação (IDE)

Os ambientes de Desenvolvidos Integrados, conhecidos como IDEs, são ferramentas que possuem recursos muitas vezes próprios e exclusivos, podendo oferecer diversas linguagens de programação, como JAVA, JavaScript, HTML & CSS, C++, entre outras diversas. As IDEs mais conhecidas são o Visual Studio Code, Eclipse, IntelliJ IDEA e Pycharm. Como já dito, cada IDE pode conter uma diferença, seja ela a formatação de código, interpretação ou a interação que o programa tem com o usuário.

Compiladores e interpretadores

Os compiladores e interpretadores podem ser compiladores ou interpretadores. Um compilador funciona "traduzindo" o código fonte para uma forma executável antes da execução do programa, fazendo com que o computador entenda o código aparecido. Já o interpretador ele lê e executa o código fonte diretamente, linha por linha. Linguagens como C, C++ e JAVA geralmente são compiladas, enquanto linguagens como Phyton e JavaScript são interpretadas. No entanto, muitas linguagens modernas possuem abordagens híbridas, onde o código é compilado just-in-time (JIT), ou seja, ele faz uma representação intermediária antes da execução.

Sintaxe e semântica

A sintaxe na programação são regras e estrutura que determinam como escrever um código válido e executável. É muito importante que o programador respeite as regras de sintaxe para que não haja nenhum erro no código. Cada linguagem de programação tem sua própria sintaxe, com palavras-chave, operadores e estruturas específicas. Erros na sintaxe podem resultar em mensagens de erro. Consultar a documentação da linguagem é importante para entender e aplicar corretamente a sintaxe.

Documentação

Na documentação de uma linguagem de programação, podemos obter informações detalhadas sobre uma linguagem de programação, bibliotecas ou ferramentas. Ela possui referências sobre a sintaxe, semântica e recursos disponíveis. A documentação é essencial para entender e compreender como usar corretamente as funcionalidades, oferece exemplos de uso do código e melhores práticas. Os programadores constantemente consultam a documentação para aprender, explorar recursos avançados e resolver problemas. É interessante que o programador veja a versão da documentação para melhores resultados.

Paradigmas de programação

Declarativos

Sobre os paradigmas declarativos, o foco está em descrever o que se deseja alcançar, em vez de se concentrar em como realizar as etapas específicas. Os programas declarativos definem as relações e restrições entre os dados, permitindo que o sistema determine a melhor maneira de executar as tarefas. Exemplos disso são SQL (para bancos de dados) e Prolog (para lógica).

Imperativos

Nos paradigmas imperativos, o foco está nas instruções detalhadas e sequenciais que especificam como executar uma tarefa. Os programas imperativos são compostos por uma sequência de comandos que modificam o estado dos dados e controlam o fluxo de execução. Exemplos de linguagens imperativas incluem C, Java e Python.

Variáveis e tipos de dados

Caracteres

Os caracteres são um tipo de dado que representa letras, números e símbolos em uma linguagem de programação. Eles são usados para representar texto em um programa e são armazenados em variáveis do tipo "string". As operações com caracteres incluem concatenação, comparação e extração de substrings. Algumas linguagens de programação oferecem recursos avançados para manipulação de caracteres, como conversão entre maiúsculas e minúsculas e substituição de caracteres específicos.

Números inteiros e reais

Números inteiros e reais são tipos de dados que podem ser usados para representar valores numéricos na programação. Os inteiros são usados para números sem parte fracionária, enquanto os reais representam valores decimais. Como por exemplo: os números inteiro, como o 1, -5, 100. Já os números reais são usados para representar valores decimais, como o 3.14, 2.5. Agora na programação como por exemplo em JAVA, o valor vazio (void) indica que uma função não retorna valor. O valor nulo (null) é usado para representar a ausência de um valor válido. Por fim, o valor indefinido é utilizado quando uma variável foi declarada, mas ainda não recebeu um valor válido.

Vazio, nula e indefinidas

Os conceitos de vazio, nulo e indefinido estão relacionados a variáveis e valores.

Vazio (void): É utilizado para indicar funções ou métodos que não possuem um tipo de retorno específico. Esse tipo de função pode ser utilizado para realizar uma determinada tarefa que não retorna nenhum valor.

Nulo (null): Representa a ausência de um valor válido em uma variável. Quando uma variável é atribuída com o valor nulo, significa que ela não está apontando para nenhum objeto ou valor.

Indefinido: Refere-se a uma variável que foi declarada, mas ainda não possui um valor atribuído. Isso pode ocorrer quando uma variável é declarada, mas não é inicializada como um valor específico. Nesse caso, o conteúdo da variável é considerado indefinido e pode conter lixo de memória ou um valor aleatório.

Booleano

Booleano é um tipo de dado encontrado na programação que pode representar os valores lógicos de verdadeiro ou falso. Ele pode ser usado para expressar condições e tomar decisões em um programa. Os valores booleanos são "TRUE" ou "FALSE" e são usados em estruturas de controle para determinar os fluxos de execução do programa com base em condições lógicas.

Funções, sub-rotinas e exceções

O que é?

Funções são como pedaços de código que você pode usar várias vezes no seu programa. Elas ajudam a organizar o código em partes menores e mais fáceis de entender. Você pode passar informações para uma função e ela pode retornar um resultado.

Sub-rotinas são parecidas com funções, mas não retornam nada. Elas só fazem uma tarefa específica quando são chamadas. Você pode reutilizar sub-rotinas em diferentes partes do seu programa.

Exceções são situações inesperadas ou erros que podem acontecer enquanto o programa está rodando. Quando uma exceção acontece, o programa para de funcionar normalmente e precisa tomar uma ação especial para lidar com o problema. Tratar exceções é importante para garantir que o programa não pare completamente quando algo dá errado.

Estrutura de dados

Cadeias de caracteres (strings)

As cadeias de caracteres, ou strings, são estruturas de dados usadas para armazenar e manipular sequências de caracteres. Elas são amplamente utilizadas para trabalhar com dados de texto em programas, assim permitindo a manipulação , busca e exibição de informações. As strings são representadas entre aspas e podem ser concatenadas, comparadas e extraídas. É importante usar as strings corretamente para garantir a integridade e a eficiência do programa.

Listas (arrays)

Listas, que também são conhecidas por arrays, são estruturas de dados que armazenam coleções de elementos do mesmo tipo. Elas são usadas para organizar conjuntos de dados, que são expressos sequencialmente. As listas tem tamanho fixo e os elementos são acessados por meio de um índice. Elas permitem adicionar, remover, pesquisar e ordenar elementos. São usadas e encontradas em várias linguagens de programação para armazenar e calcular dados de forma mais estruturada.

Tuplas

As tuplas são coleções ordenadas e imutáveis de elementos. São utilizadas para agrupar dados relacionados que não precisam ser modificados. Cada elemento poderá ter um tipo diferente. As tuplas são definidas entre parênteses e os elementos são separados por vírgulas. Sua mutabilidade garante a integridade nos dados. Podem ser usadas em operações como atribuição, acesso e desempacotamento. Também podem ser usadas como chaves em dicionários. Simplificando tudo, as tuplas são estruturas de dados imutáveis para agrupar dados relacionados.

Dicionários

Dicionários podem ser descritas como estruturas de dados que armazenam pares de chave-valor. Eles permitem associar valores a chaves únicas para uma recuperação rápida dos dados. Os dicionários são utilizados para representar as informações relacionados, onde cada elemento possui uma identificação única. Eles oferecem muita flexibilidade na dição, modificação e remoção de elementos. Os dicionários são amplamente usados em várias linguagens de programação.

Conjuntos

Conjuntos podem ser definidas como estruturas de dados que armazenam elementos únicos e não ordenados. Eles são úteis para remover duplicatas e realizar operações matemáticas comuns. Os conjuntos são implementados em várias linguagens de programaçãop

Estruturas de controle

De seleção

Estruturas de controle podem ser expressas como recursos usados na programação para controlar o fluxo de execução de um programa. Elas podem incluir estruturas de decisão como o "if e "else", que permitem tomar decisões com base em condições, estruturas de exceções, como o "for" e "while", que permitem repetir blocos de códigos, e estruturas de controle de exceção, como o "try-catch", que lidam com erros e exceções durante a execução. Essas estruturas são essenciais para criar lógica e controlar o seu programa.

Testes

Unitários

Os testes unitários podem ser descritos como testes automatizados que verificam o funcionamento correto de pequenas partes de código, conhecidas como unidades. Eles são escritos pelos desenvolvedores e executados de forma independente para garantir a qualidade do software. Os testes cobrem diferentes cenários e ajudam a detectar erros precocemente, facilitar a refatoração do código para melhorar a qualidade do software. Em resumo, os testes unitários são uma prática essencial para garantir a durabilidade e confiabilidade do código.

Integrados

Os testes integrados, também conhecidos como testes de integração, são uma técnica de teste de software em que diferentes módulos ou componentes de um sistema são combinados e testados como um todo. Esses testes verificam se as interações e comunicações entre os componentes estão funcionando corretamente, garantindo a integridade e a correta funcionalidade do sistema como um todo. Os testes integrados podem envolver a interação de diferentes camadas do software, como a interface do usuário, a lógica de negócios e o banco de dados. Eles são realizados para identificar possíveis problemas de integração e garantir que o sistema funcione de maneira consistente e sem erros.

JAVA

é uma linguagem de programação de alto nível, orientada a objetos e multiplataforma. É utilizada para desenvolver aplicativos de desktop, sistemas web, aplicativos móveis, entre outros. O Java é conhecido por ser seguro, confiável e escalável, o que o torna uma escolha popular para empresas que desenvolvem aplicativos corporativos.

JavaScript

JavaScript é uma linguagem de programação interpretada, leve e orientada a objetos, utilizada para adicionar interatividade em páginas da web. É amplamente empregada na criação de animações, menus dinâmicos e validações de formulários. Executado no lado do cliente, ou seja, no navegador do usuário, o JavaScript é uma das tecnologias mais populares para o desenvolvimento front-end.

HTML

é uma linguagem de marcação usada para criar páginas da web. Com HTML, é possível definir a estrutura e o conteúdo de uma página, como textos, imagens, links e formulários. É uma linguagem relativamente simples, mas fundamental para o desenvolvimento web.

CSS

é uma linguagem de estilo usada para definir a aparência e o layout de uma página da web. Com CSS, é possível personalizar a aparência de elementos HTML, como fontes, cores, margens e posicionamento. O CSS é frequentemente usado em conjunto com o HTML e o JavaScript para criar páginas da web mais complexas e interativas.

Projetos Desenvolvidos

Projeto Gerador de Página HTML e Js

Neste projeto nós desenvolvemos em Squad's um programa que ao adicionar o HTML e JavaScript ele gerava uma página pré-formatada com estilização já pronta feita no CSS. Usamos os exemplos de código dos exercícios de JavaScript passados pelo professor Cainã, quando ainda estávamos a aprender a linguagem.

Como Construímos este projeto?

Neste projeto utilizamos as linguagens HTML, CSS, JavaScript. Começamos planejando como ficaria este site, montamos um rascunho de estilização usando o figma, e separamos em grupos, no qual pudemos dividir funções diferentes para cada dupla, uma dupla ficava com o JavaScript e com outra dupla ficava o HTML e CSS.

HTML:

Começamos implementando o HTML em nosso site, para estrutura-lo. O HTML é muito importante, pois com ele conseguimos ligar o CSS e o JavaScript por meio de "classes" e "IDs", como por exemplo, o botão gerar o site é conectado com uma "ID", como por exemplo: <button id="gerarPagina">, costumamos deixar as ID's da maneira mais fácil de ser encontrada pelo seu parceiro de trabalho, é um bom costume da programação. No entanto em nosso projeto utilizamos os <input> e os <textarea>, utilizamos o <textarea> para inserir as linguagens de programação, e o <input> para declararmos como botão (O input é um elemento HTML que permite aos usuários inserir dados interativamente em um formulário ou interface).

JavaScript:

No JavaScript usamos uma função de esconder a página e sobrepor a outra página, por exemplo a primeira página tem a id = page1 e a segunda página id = page2, na segunda parte podemos ver que ele permite que o usuário insira código HTML em um campo de entrada e exiba-o em uma área específica da página quando um botão for clicado.

Por fim, no final temos uma função de dark-mode e white-mode, que deixa a página com um tema escuro ou claro. Ela é criada percorrendo todo o body do HTML, para em fim mudar a cor do fundo.

CSS:

No CSS fornecemos pré-definições de estilização para todos os botões. Além disso, definimos cores para nossa tela de fundo e para o bloco de texto. E com a função de mudar de tema do claro para o escuro, tivemos que apresentar os botões e blocos de texto como IDs, pois dessa forma podemos selecionar o botão que quisermos, e não apenas englobar todos os elementos em uma classe.
